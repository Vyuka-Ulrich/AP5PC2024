### How to run on windows:
```bash
$ mkdir build
$ cd build
$ cmake -G "MinGW Makefiles" ..
$ cmake --build .
```

### How to run on unix:
```bash
$ mkdir build
$ cd build
$ cmake -G "Unix Makefiles" ..
$ cmake --build .
```
